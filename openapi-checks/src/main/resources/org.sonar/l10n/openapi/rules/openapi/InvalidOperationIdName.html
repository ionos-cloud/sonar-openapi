As defined in the SDK guidelines, the operationId SHOULD be defined in the form path+crud. Keeping a similar operationId definition for all swaggers helps maintain consistency and ease-of-use. See https://docs.google.com/document/d/1DZea04mLSH0pqZRrHQxHsOrAxu4J3XfA/edit#heading=h.17dp8vu

Non-compliant Code Example

 "/clusters/{clusterId}/snapshots":
        get:
            operationId: retrieveClusterSnapshots

Compliant Solution

 "/clusters/{clusterId}/snapshots":
        get:
            operationId: clustersSnapshotsGet
