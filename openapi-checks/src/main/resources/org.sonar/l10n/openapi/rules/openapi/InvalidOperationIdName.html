As defined in the SDK guidelines, the operationId SHOULD be defined in the form path+crud. Keeping a similar operationId definition for all swaggers helps maintain consistency and ease-of-use. See https://github.com/OAI/OpenAPI-Specification/blob/master/versions/2.0.md#path-item-object.

Non-compliant Code Example

 "/clusters/{clusterId}/snapshots":
        get:
            operationId: retrieveClusterSnapshots

Compliant Solution

 "/clusters/{clusterId}/snapshots":
        get:
            operationId: clustersSnapshotsGet
